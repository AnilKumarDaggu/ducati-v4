## Scope

- The experience constitution: STUDIO / OFFICINA / TECNICO spatial language
- Visual design system ("Tavolozza Officina"): color, typography, 3D material library tokens
- Camera & interaction grammar (AAA spec: damped orbit, framing engine, hover language)
- Motion & animation guidelines (timing scale, easing tokens, Mechanical Honesty Principle)
- Per-module experience direction (Explorer, Build Simulator, Assembly, Manufacturing)

## Relationship to other categories

Engineering documents (ARCH, SIM, MFG) define *what the platform does and that it is correct*; UXD-001 defines *what it feels like and that it is worth using*. Both are release gates (UXD-001 Part III adds the "side-by-side test" to Sprint 2 acceptance).
