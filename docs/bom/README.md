# Bill of Materials

Part hierarchies, component attribution, and digital-twin modeling scope.

| ID | Title | Version | Status |
|---|---|---|---|
| [DTEA-BOM-001](DTEA-BOM-001-hierarchical-bom-v1.0.md) | Hierarchical BOM (Levels 1–6, ~1,850 unique parts est.) | 1.0 | Approved |

## Scope

- 6-level part hierarchy (motorcycle → systems → subsystems → assemblies → components → parts)
- Attribute coding standard (manufacturing, animation, explode, cutaway, simulation codes)
- Four fully attributed exemplar expansions (cylinder head, cranktrain, brake caliper, radar group)
- Part count estimates and modeling tiers
